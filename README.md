# Arpit Verma — Personal Portfolio Website

## Overview

This is the source code for my personal portfolio website, designed and developed to showcase my academic journey, core skills, and personal projects.

The website provides an engaging and responsive platform for recruiters and peers to learn about my background as an aspiring AI/ML Engineer and Tech Enthusiast.

## Key Features

The portfolio is structured into the following sections:

* **Home:** A brief introduction, greeting, and call-to-action to hire me.
    * **Developer:** Arpit Verma
    * **Role:** B.Tech — CSE (AIML) | 1st Year at KIET College, Ghaziabad.
    * **Tagline:** Aspiring AI Engineer | Tech Enthusiast.
* **About Me:** Detailed background on my passion for Artificial Intelligence, Machine Learning, and Web Development.
* **Skills:** A visual representation of my technical competencies.
    * **Core Skills:** HTML (90%), CSS (85%), C Language (70%).
* **Projects:** Showcase of development work. (Currently listing a 'travel website' project).
* **Contact:** A dedicated section with a contact form and links to my social profiles (GitHub, Instagram, Email).
* **Design:** Clean, modern, and responsive design with a striking **Red and Black** color scheme.
